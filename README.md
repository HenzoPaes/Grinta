# Grinta 🔥

> **Grinta** *(italiano: garra, determinação, espírito de luta)*
>
> Multiferramenta de produtividade offline: **Projetos · Conversor · Finanças · Paletas · Timeline · Mapa Mental**

---
# O que é?

Grinta é um app desktop feito com tecnologias web embaladas em runtime nativo para rodar como aplicativo (.exe / .dmg / .AppImage). Ideal para quem quer organizar projetos, conter conversões rápidas, controlar finanças simples e rabiscar mapas mentais — tudo offline.
